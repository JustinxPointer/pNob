# 🚀这是一个在线笔记小项目，采用了MySQL+Flask+html/css的前后端分离架构，采用了uv来管理依赖，适合在本地环境部署、使用与二次开发学习
## 🛠️📝安装指南
1.克隆本仓库到本地\
```pass```\
2.安装uv\
```pass```\
3.进入项目文件夹\
```uv sync```\
4.连接mysql,将notebook.sql导入即可，然后根据你的mysql配置更改utili文件中的user和password，默认root用户不用更改\
5.一键运行本项目\
```uv run manage.py```
## 贡献代码
- 👀💪🏻首先Fork本仓库，clone项目后进入项目目录,安装必要目录,推荐使用uv管理依赖
- ‼️不要在main分支上修改，使用
- ```git checkout -b feature/你的fix ```
- 创建新的分支后进行更新⬆️，最后Pull Request
